The takitapart CSS Framework

A semantic, fluid, responsive framework cobbled together from several awesome projects by people much more clever than myself.

Built for me, shared with you

Check it out at the takitapart-framework GitHub page.

it wouldn't be possible without

The takitapart framework is largely adapted from Joni Korpi's fantastic Golden Grid System, with several SASS mixin's to do the heavy math. Additional code from the Normalize.css and HTML5 Boilerplate projects was also used.
